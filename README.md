## Installation

```bash
  npm run server
  npm run
```
    
## Running Tests

rodar todos os testes de uma vez
```bash
  npm run test
```

rodar todos os testes de uma pasta 
```bash
  npm test src/__tests__/components/Header.spec.tsx   
```

rodar todos testes de um arquivo
```bash
npm run teste -- "useCart"
```
